# 史実カード レアリティ設計

## 1. 目的

Learning RPG の史実カードには、通常のカテゴリ分類に加えて、レアリティを持たせる。

特に、国宝・重要文化財級の文化財や、その時代を象徴する史実要素は **Sランクカード** として扱う。

これにより、カード収集にゲーム的な高揚感を持たせつつ、歴史・文化の重要度も自然に伝えられる。

## 2. 基本方針

レアリティは、単なる入手難度ではなく、以下を表す。

- 歴史的な重要度
- 文化的な価値
- その時代を象徴する度合い
- 他のカードとの接続の多さ
- プレイヤーが見つけたときの発見感

ただし、レアリティ付けも史実的な確からしさを優先する。

見た目が派手だからSランクにするのではなく、歴史的・文化的に重要なものをSランクにする。

## 3. レアリティ案

### Sランク：時代を象徴する特別カード

国宝・重要文化財級、またはその時代を代表する文化・建物・人物・事件など。

#### 例

- 飛鳥寺に関わる重要要素
- 仏像
- 木簡の重要資料
- 法隆寺に関わる建築・仏像・美術要素
- 推古天皇、聖徳太子、蘇我馬子などの重要人物
- 大化の改新につながる政治的要素

※個別に採用する際は、必ず史実根拠を確認する。

### Aランク：中核カード

時代理解に欠かせないカード。

町歩き・クエスト・記録帳の中心に使える。

#### 例

- お寺の瓦
- 寺をつくる職人たち
- 木簡
- 僧
- 役人
- 豪族の屋敷
- 市の荷物

### Bランク：生活・背景カード

時代の生活感を支えるカード。

中核ではないが、町の解像度を上げる。

#### 例

- 素朴な器
- 木材を運ぶ人々
- 職人の道具
- 市で扱う荷物
- 服装の一部
- 履き物

### Cランク：演出候補カード

面白いが、史実根拠や扱いに注意が必要なカード。

監修前は中核にしない。

#### 例

- 甘い香りの屋台
- 飛鳥のお菓子
- 具体的な子どもの遊び
- 庶民の食事の細部

### Dランク：不採用候補

時代錯誤、または誤解を招く可能性が高いカード。

原則として使わない。

#### 例

- 江戸風の団子屋台を飛鳥時代の市場に置く
- 砂糖菓子を飛鳥の庶民のおやつとして扱う
- 確認できない遊びを「飛鳥時代の代表的な遊び」とする

## 4. Sランクカードの条件

Sランクカードは、以下のうち複数を満たすものに限定する。

- 国宝・重要文化財など、現代でも高く評価されている文化財に関わる
- その時代を象徴する
- 教科書的にも重要だが、ゲーム内では説明ではなく体験として出せる
- 複数カテゴリに接続できる
- 他時代とのつながりを生みやすい
- 記録帳で見返したくなる

## 5. Sランクカードの扱い方

Sランクカードは、簡単にばらまかない。

### 獲得方法

- 章の重要イベントで出会う
- 複数の関連カードを集めると解放される
- 町の探索を進めることで姿が見える
- 直接入手ではなく、最初は「未完成」「遠景」「断片」として見せる

### 表示演出

- 通常カードより少し特別な演出
- ただし派手すぎない
- 国宝・文化財をゲーム的に軽く扱いすぎない
- 説明は短く、詳細はカード裏面に置く

## 6. Sランクカードと時代間リンク

Sランクカードは、時代間リンクの中心になる。

例：

- 飛鳥の仏教文化 → 奈良の大仏・寺院文化 → 鎌倉仏教 → 室町の禅文化
- 飛鳥の瓦・寺院建設 → 奈良の大寺院 → 後世の寺社建築
- 木簡・記録 → 律令国家 → 文書行政 → 近代の新聞・記録文化

Sランクカードは単なるレアカードではなく、時代の流れをつなぐハブとして扱う。

## 7. Chapter 1 飛鳥の町でのSランク候補

### 候補

- 飛鳥寺に関わるカード
- 仏像に関わるカード
- 法隆寺に関わるカード
- 聖徳太子カード
- 推古天皇カード
- 蘇我馬子カード
- 大化の改新につながる政治的カード

### 注意

Chapter 1 のMVPでは、Sランクカードを大量に出さない。

まずは、以下のような運用がよい。

- Sランク候補を遠景や断片として見せる
- 初回はAランクカードを中心に集める
- 章の終盤でSランクカードの存在を示す
- Sランクカードの本格取得は、関連カードが集まってからにする

## 8. ゲーム体験としての意味

通常カードは、町の生活を知るためのカード。

Sランクカードは、その時代の核心に触れるカード。

プレイヤーは、最初からSランクだけを追うのではなく、生活カード・道具カード・人物カードを集めることで、少しずつSランクカードの意味が分かるようになる。

## 9. 重要な設計思想

> Sランクカードは、ごほうびではなく、時代の核心である。

国宝級のものをただのレア演出にしない。

そこに至るまでの生活・技術・人物・文化のカードを集めることで、プレイヤーが「なぜこれが重要なのか」を自然に感じられるようにする。
