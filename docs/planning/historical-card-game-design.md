# 史実カードゲーム設計

## 1. 目的

Learning RPG は、史実カードを単なる図鑑・記録帳として集めるだけでなく、カードゲームとしての面白さを持たせる。

ただし、このゲームにおけるカードゲーム性は、架空効果やファンタジー能力を競うものではない。

史実カードを集め、つなげ、使うことで、時代の構造・文化の関係・出来事の背景が見えてくる体験を作る。

## 2. 基本コンセプト

> 史実カードを集める。  
> 史実カードをつなげる。  
> 史実カードを使って、町の出来事を読み解く。

Learning RPG のカードゲーム性は、暗記クイズではなく、**関係性を発見する遊び**に置く。

## 3. カードゲームとしての面白さ

### 3.1 集める面白さ

町を歩き、物を見る、人と話す、手伝うことでカードが増える。

例：

- お寺の瓦
- 寺をつくる職人たち
- 木簡
- 役人
- 僧
- 市の荷物
- 供物
- 飛鳥寺

カードが増えるほど、その時代の町の解像度が上がる。

### 3.2 つなげる面白さ

カードは単体で終わらず、関係カードとリンクする。

例：

- お寺の瓦 → 寺をつくる職人たち → 飛鳥寺
- 木簡 → 役人 → 市の荷物
- 供物 → 僧 → 儀式 → 飛鳥寺
- 蘇我馬子 → 飛鳥寺 → 仏教文化

プレイヤーは、カード同士の線がつながることで、時代の構造を理解する。

### 3.3 使う面白さ

集めたカードを、町の出来事を進めるために使う。

ただし、カードバトルではない。

使い方は以下。

- NPCに見せる
- 場所で使う
- 関係カードを組み合わせる
- 手がかりとして提示する
- 町の困りごとの背景を読み解く

例：

> 寺の人が市へ取りに来ない。  
> プレイヤーは「市の荷物」「供物」「僧」「お寺の瓦」などのカードを集める。  
> それらをつなげることで、寺で儀式準備と建設作業が重なって混乱していることが見えてくる。

## 4. カードの基本構造

各カードは、以下の情報を持つ。

- カード名
- 時代
- カテゴリ
- レアリティ
- 史実確度
- 表面説明
- 裏面説明
- 見つけた場所
- 関連カード
- 使用できる場面
- 時代間リンク

## 5. カードカテゴリ

### 5.1 人物カード

実在人物・役割人物の両方を扱う。

例：

- 聖徳太子
- 推古天皇
- 蘇我馬子
- 僧
- 役人
- 寺をつくる職人たち
- 工人

### 5.2 建物カード

町の構造や時代の象徴を表す。

例：

- 飛鳥寺
- 法隆寺西院伽藍
- 石舞台古墳
- 豪族の屋敷
- 市

### 5.3 道具・技術カード

時代の手触りを作る。

例：

- お寺の瓦
- 木簡
- 土器
- 筆
- 職人の道具

### 5.4 食カード

暮らしや儀式に接続する。

例：

- 供物
- 米・雑穀
- 果実・木の実
- 素朴な器に盛られた食事

### 5.5 服装カード

時代差が一目で分かるカード。

例：

- 僧の衣
- 役人の服
- 職人の作業着
- 冠
- 庶民の衣

### 5.6 事件・制度カード

時代の大きな変化を表す。

例：

- 冠位十二階
- 十七条憲法
- 遣隋使
- 乙巳の変
- 大化改新

### 5.7 戦争・争いカード

後続章で扱う重めのテーマ。

最初のMVPでは扱いすぎない。

## 6. レアリティ設計

### Sランク

時代の核心カード。

国宝・重要文化財級、または時代を象徴する人物・建物・事件。

例：

- 飛鳥寺
- 飛鳥大仏
- 法隆寺西院伽藍
- 聖徳太子
- 推古天皇
- 蘇我馬子

### Aランク

章の中核カード。

例：

- お寺の瓦
- 寺をつくる職人たち
- 木簡
- 役人
- 僧
- 供物

### Bランク

生活感・背景カード。

例：

- 素朴な器
- 木材を運ぶ人々
- 職人の道具
- 市の荷物
- 服装の一部

### Cランク

演出候補。

史実確認が済むまで中核にはしない。

例：

- 甘い香りの屋台
- 飛鳥のお菓子
- 具体的な子どもの遊び

## 7. カードの入手方法

カードは、クイズの正解ではなく、体験によって手に入る。

### 入手パターン

- 見る：対象を発見する
- 聞く：NPCに話を聞く
- 触れる：調べる、近づく
- 手伝う：短いイベントを解決する
- つなげる：関連カードを複数集める

### 例：お寺の瓦

入手条件：

1. 寺へ向かう途中で瓦を運ぶ職人を見る
2. 瓦を調べる
3. 職人から「お寺の屋根に使う」と聞く

### 例：飛鳥寺

入手条件：

1. 寺に到着する
2. 「お寺の瓦」「寺をつくる職人たち」「僧」などの関連カードを集める
3. 章の重要イベントでSランクカードとして解放される

## 8. カードの使い方

### 8.1 NPCに見せる

カードを持っていることで、NPCとの会話が変わる。

例：

- 職人カードを持っていると、寺の作業場で追加会話が出る
- 木簡カードを持っていると、役人の話が少し分かる

### 8.2 場所で使う

特定の場所でカードを使うと、見えるものが増える。

例：

- 「お寺の瓦」を持って寺へ行くと、屋根や建設作業に注目できる
- 「供物」を持って寺へ行くと、儀式準備に気づける

### 8.3 組み合わせる

カード同士を組み合わせることで、関係が見える。

例：

- 供物 + 僧 + 飛鳥寺 = 儀式の準備
- 木簡 + 役人 + 市の荷物 = 物資管理
- 瓦 + 職人 + 工人 = 寺院建設の技術

注意：

カードの組み合わせは、史実的に確認された関係だけを許可する。

プレイヤーが自由に捏造できるシステムにはしない。

## 9. デッキの考え方

Learning RPG では、通常のカードバトルのようなデッキ構築は必須にしない。

ただし、章が進むと「持ち歩くカード」を選ぶ仕組みを検討できる。

### 初期MVP

- デッキ構築なし
- 集めたカードはすべて記録帳に入る
- 必要な場面で自動的に関連カードが参照される

### 将来案

- 町に入る前に、注目するカードセットを選ぶ
- 文化デッキ、食デッキ、人物デッキ、技術デッキなどを組める
- 選んだカードによって、見える会話や注目ポイントが変わる

例：

- 技術デッキ：瓦、職人、工人、木簡
- 文化デッキ：僧、供物、仏像、飛鳥寺
- 人物デッキ：蘇我馬子、推古天皇、聖徳太子、役人

## 10. カードゲームとしての章進行

### 10.1 章の序盤

生活・道具・人物のB/Aカードを集める。

プレイヤーはまず、町の手触りを知る。

### 10.2 章の中盤

カード同士の関係がつながり始める。

例：

- 市の荷物と供物が寺へつながる
- 瓦と職人が寺院建設へつながる
- 木簡と役人が町の管理へつながる

### 10.3 章の終盤

Sランクカードが解放される。

例：

- 関連カードを集めたことで「飛鳥寺」カードの意味が分かる
- 飛鳥寺カードが、後の奈良・鎌倉・室町の寺院文化へのリンクになる

## 11. カードゲーム性の合格ライン

- カードを集めること自体が楽しい
- カード同士の関係を見るのが楽しい
- レアカードが単なる報酬ではなく、時代理解の節目になっている
- 子どもでも直感的に触れる
- 大人が見ても史実の構造が面白い
- クイズに頼らず学びが成立する

## 12. 失敗条件

以下は避ける。

- カードがただのコレクションになっている
- Sランクカードだけ集めればよくなる
- 史実的に弱いカードが目立つ
- カード効果がファンタジー化しすぎる
- 「正解カードを選べ」というクイズになる
- カードの説明文が長すぎて教材になる
- カードの組み合わせで誤った歴史理解を生む

## 13. Chapter 1での初期カードゲームループ

### 流れ

1. 市の一角で「市の荷物」「供物」に出会う
2. 寺の人が来ないと知る
3. 寺へ向かう途中で「お寺の瓦」「寺をつくる職人たち」を集める
4. 役人や木簡に出会い、「木簡」「役人」を集める
5. 寺に着き、「僧」と出会う
6. 関連カードがつながり、Sランク「飛鳥寺」の輪郭が見えてくる

### 最初のSランク演出

Chapter 1の終盤で、飛鳥寺カードを解放する。

ただし、いきなりカードだけ出すのではなく、関連カードが集まったことで、プレイヤーが「この寺は、いろんな人と物が集まってできている」と感じられる状態にする。

## 14. 重要な設計思想

> 史実カードゲームの面白さは、強いカードで勝つことではない。  
> バラバラに見えた人・物・建物・食・制度がつながり、時代の姿が見えてくることにある。

カードゲーム性は、歴史理解のためのエンジンである。
