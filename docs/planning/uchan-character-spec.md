# うーちゃん キャラクター仕様メモ

## 1. 位置づけ

うーちゃんは、Learning RPG に登場する小さな案内役であり、主人公の相棒である。

モチーフは、ユーザーが飼っているホオミドリアカオウロコインコ。

教師やナレーターではなく、主人公と一緒に時代の町を見て、驚き、気づき、少しだけ背中を押す存在として扱う。

## 2. 基本役割

- 夢の中の「時の回廊」で主人公と出会う
- 時代の町への入口を案内する
- 子どもの疑問に寄り添う
- 気になる場所へ飛んでいく
- 町の中で見つけたものに軽く反応する
- 記録帳の相棒になる
- ただし、答えを全部説明しない

## 3. 話し方

### 方針

話し方は、**A寄りのB**。

- ベースは「元気で子どもっぽい相棒タイプ」
- 必要な場面では、少しだけ案内役として振る舞う
- 説明役に寄せすぎない
- 発見したものへのリアクションで場を動かす

## 4. 口調イメージ

### 基本

- 明るい
- 好奇心旺盛
- やさしい
- 少しおしゃべり
- 食べ物や光るものに反応しやすい

### セリフ例

- 「わっ、いいにおい！あっちからするよ！」
- 「ここ、ふつうの町とちょっと違うね。」
- 「うーちゃん、あの布がゆれてるところ、気になる！」
- 「見たことない食べものだね。どこから来たんだろ？」
- 「まずは市場を歩いてみよう。なにか見つかるかも！」

## 5. やらないこと

- 歴史用語を長く説明しない
- 正解を教える先生にならない
- プレイヤーを急かしすぎない
- クイズの出題者にならない
- 物語の主役を奪わない

## 6. うーちゃんの良い使い方

うーちゃんは、プレイヤーに「考えさせる」のではなく、「気づきたくなる」ようにする。

例：

- 香りがする方向へ飛ぶ
- 見慣れない道具の近くで首をかしげる
- 市場のにぎわいに反応する
- 子どもNPCと一緒に驚く
- 記録帳に新しいカードが増えたときに喜ぶ

## 7. 初回登場イメージ

現代の子どもが夢の中で、絵本のような時の回廊に迷い込む。

やわらかい光の中から、うーちゃんが飛んでくる。

うーちゃんは、主人公を怖がらせず、明るく声をかける。

> 「おはよう？ それとも、こんばんは？ ここは時の回廊だよ！」

この時点では、時代や歴史について詳しく説明しない。

まずは、最初の町である飛鳥の町へ自然に誘導する。
